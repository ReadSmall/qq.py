``qq.ext.commands`` -- 机器人命令框架
====================================================

``qq.py`` 提供了与 qq 交互的较低级别的方面。 
通常，该库用于创建机器人。 
然而，第一次正确完成这项任务可能会令人生畏和混乱。 
很多时候，在创建一个可扩展、灵活和强大的机器人命令框架时会出现重复。 
出于这个原因，``qq.py`` 带有一个扩展库，可以为你处理这个问题。
