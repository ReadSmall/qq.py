欢迎来到 QQ.py 的文档
==================

**qq.py** 是用于 QQ 频道的现代化、易于使用、功能丰富且异步就绪的 API Wrapper。

**Features:**

- 使用 ``async``\/``await`` 语法的现代化 Pythonic API
- 帮助创建机器人的命令扩展
- 易于使用的面向对象设计
- 针对速度和内存进行了优化

.. note::

   该项目正在积极开发中.
